HorseGPT 使用条款

根据《生成式人工智能服务管理暂行办法》，中华人民共和国境内用户请使用国产大模型，请勿用于非法行为和操作。本服务服务器位于日本，负载服务器位于美国，请遵守所在地和相关地区的法律法规。

提供的服务

HorseGPT
包括自主训练的大语言模型 HorseGPT，以及多种经过二次训练及优化的开源大模型，包括 Llama、Llava 等。
HorseGPT API
提供多种大模型的 API 服务，并提供用户面板、令牌发放等服务，便于用户控制 AI 的使用。。
API 转发与加速服务
由 Cloudflare 提供，使全球大多数国家能够快速访问我们的 API。
HorseGPT Chat
提供接入 API 的基础聊天、文本生成及 AI 使用服务。HorseGPT Chat 需要通过密码或 HorseGPT API 令牌才能访问。

常见问题

我该如何注册并使用该服务
请参考使用指南。
我无法在 HorseGPT API 系统中获取兑换码并充值
由于测试所限及相关地区对测试的法律法规限制，本服务并未开通稳定的兑换码获取渠道。您可以邮件联系：admin@nu.ac.cn了解更多信息。
我无法在 HorseGPT Chat 中使用我的令牌
请确认余额及令牌的可用性。请确认您在 HorseGPT Chat 中输入的令牌是否正确，并在正确的位置输入。
我在使用 API 服务 / HorseGPT Chat 时出现了报错
如果是网站或系统问题，请通过此链接提交工单。如果 AI 回复时报 400、 403 或上游渠道错误，这是由于上游渠道限额所致，请稍后重新使用，如果问题持续或反复出现，请通过邮件：admin@nu.ac.cn或工单链接及时联系我们。
